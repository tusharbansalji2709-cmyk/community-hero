## 📋 Project Description (for Google Doc submission)

**Problem Statement Selected:** Community Hero — Hyperlocal Problem Solver

**Solution Overview:**
CommunityHero is an AI-powered civic issue reporting platform where citizens can report, verify, and track community problems (potholes, water leaks, broken lights, waste issues) in their neighborhood. A live Gemini AI agent autonomously categorizes each report, assesses severity, routes it to the right municipal authority, and notifies nearby citizens — removing bureaucratic friction from the civic complaint process.

**Key Features:**
- Gemini Vision AI auto-analyzes uploaded photos to identify issue type and severity
- Autonomous agent routes issues to correct authority (BMC, MCGM, MSEDCL) with SLA
- Community verification system with gamified XP points and leaderboard
- Real-time issue map with geo-clustered pins by category
- Predictive insights powered by Gemini (monsoon patterns, cluster detection)
- AI chat assistant for guided reporting

**Technologies Used:**
- Frontend: Vanilla HTML/CSS/JavaScript (single file, zero dependencies)
- AI: Google Gemini 1.5 Flash (Vision + Text)
- Hosting: Google Cloud via AI Studio

**Google Technologies Utilized:**
- Gemini API (generativelanguage.googleapis.com)
- Google AI Studio (deployment)
- Google Fonts API
